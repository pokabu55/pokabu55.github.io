フォトウォーク＠浜離宮

浜離宮へお花見に行ってきたので、写真を貼っていく



浜離宮とは

  • 徳川将軍家の庭園、とパンフに書いてあった
  • 正式名称は、浜離宮恩賜庭園 というらしい
  • 場所は、汐留の近くです


  • 自分は車で行きましたが、首都高が運営している汐留駐車場に置きました。210円/30分と周辺に比べるとかなりの格安で、駐車券を浜離宮のチケット売り場で出すと、30分の割引サービスが受けられました
  • 公式サイト
  • wikipedia

フォトウォーク

お花たち

  • さくら、もも、菜の花などが見れました
  • さくらを見に行ったんですが、思ったよりさくら（ソメイヨシノ）の樹が少なかったなぁ…
  • あ、引いた画を撮るのを忘れてる!!
  • アップで！ ぼかして！ ばかりで頭がいっぱいだった。

  • ソメイヨシノ１：白っぽいので、露出を合わせるのが難しい時もあった


  • もも１：ソメイヨシノよりも色が濃くて合わせやすかった


  • 菜の花：庭園内に菜の花畑があって、たくさん咲いてた。バックは近隣のビル。


動物たち

  • とりさん１：名前は何だろう？こいつらは必死に餌を探してた。常に忙しそうにしてる。


  • 水どりさん１：水どりたちは、基本、水の上で寝てばかり。お腹いっぱいなの？潜る瞬間をとらえた！と思ったが、ピントが合ってない。


  • ねこさん１：庭園内に住んでるの？ひなたぼっこで毛づくろい。


  • ミツバチ：こんな写真撮れると思わんかった。脚に黄色の花粉団子が！！！


お茶屋

  • 庭園でのお花見といえば、お茶屋さんでしょ！！



  • 抹茶と和菓子のAセットを食べました


建築物

  • 庭園のすぐ外には高層ビルが。


  • お茶屋からの風景


  • 勝どき橋？かな？


フォトウォークを終えて

良かった点

  • 花、動物、食べもの、などなど、多くのものを撮影できた

反省点

  • 庭園に入った頃は晴れたり、曇ったりで、露出の設定が難しかったり、変更を忘れてしまったり
  • 望遠レンズと標準レンズの使い分けが、だんだんと面倒に…
  • 引きの写真はスマホにしてしまった

感想

  • 思ったよりのソメイヨシノが少なかった…。
  • やっぱり、望遠レンズと標準レンズの使い分けが、難しいな
  • 以上。

ホワイトバランス比較実験

ホワイトバランスをいろいろ変えて撮影し、比較してみた

動機

  • 夕焼けはピンク派？オレンジ派？というツイートを見て、気になったので試してみることに。
  • 正直これまでは「オレンジでしょっ！」といった固定観念があったので驚いた。
  • お昼頃までは雨まじりの曇天だったので今日の撮影は諦めてたが、夕方になり晴れてきたので急遽結構！

撮影対象と設定

  • 対象は夕焼け
  • 使用機材はもちろん、D3400！
  • ホワイトバランスのみを変え、それ以外は同じ設定で撮影（三脚不使用）
  • 基本設定：
    • Aモード、絞り開放になってた…（ま、今回はWBの実験ということで）
    • 露出±０
  • D3400のホワイトバランス設定
    • 電球、蛍光灯、晴天、フラッシュ、曇天、晴天日陰



比較実験：日没前と日没後の２つの実験を実施

実験１：全６設定で比較（日没前）

  • 電球：予想通りに青っぽい

  • 蛍光灯：意外にもピンクっぽい！！！

  • 晴天：以下４つの大きな違いは分からず…

  • フラッシュ

  • 曇天

  • 晴天日陰：敢えて言うなら、オレンジが強い？

実験２：電球、蛍光灯、晴天日陰で比較（日没後）

  • 実験１で違いが出てた電球、蛍光灯、晴天日陰の３設定での比較

  • 電球：青っぽいですね

  • 蛍光灯：ピンク！

  • 晴天日陰：安定のオレンジ。

実験を終えて

良かった点

  • ググれば、たくさんの記事が出てくるので当然の結果だが、自分で確かめられたのは良かった。
  • きっかけは、とあるツイートだったが、「夕焼けはオレンジ」が自分の固定観念であったことに気づけた。
  • 比較してみると、ピンクっぽい仕上がりが気に入ったかも！！

反省点

  • 客観的な評価指標を使えれば、もっと分かりやすかったなぁ。
  • 設定を書き出して気づいたが、絞りって今までは余り気にしたことがなかった。
  • こういった風景の撮影の時は、もう少し絞ったほうがいいのかな？とも。
    • 夕方だと周辺が暗くなる。
    • シャープでなくなる。
    • 背景をぼかすわけでもないので、被写界深度を浅くする必要ないか。
    • 参考：photo miniさんの絞りの解説
    • 基本 F8 くらいで良かったのかな、と。

感想

  • 並べて比較してみると、なるほど分かりやすい結果を得られた。
  • 説明によっては、以下のように書かれていることも多く、ピンクという結果は意外だった。
    • 電球 <- -> 曇り
    • 青 <- -> 赤
  • 何でも撮影してみないと、わからないもですね。

  • 最後に上とは別の 蛍光灯 設定の写真を載せます。

  • 以上。

満月！

満月を撮ってみました

  • 日没後に見えたので撮ってみた
  • iPhone に入れてる Magic Hour というアプリで通知設定しておくと、便利です。
  • 月の大きさが伝わるように、建物も一緒に収まるようにしました。（満月なのにサンドラッグ…）



撮影を終えて

設定

  • SS優先モードとした。1/500。
  • ISOは3200。眩しすぎるぜ!! 満月。
  • WBは自動。
  • 露出は±０だった。かな？
  • 三脚で撮影。

良かった点

  • とっさに撮ったにも関わらず、ある程度、思ったとおりに撮れた。

反省点

  • 設定をいろいろ変えて撮ってみようと思ってる間に、どんどん月が昇ってしまった。
  • 時間が経つと月が地上から離れてしまい、一緒に収めることがむずかしくなった。
  • 瞬時の判断が必要。これは、朝焼けや夕焼けの撮影でも同じ。

感想

  • 設定に迷うと、「撮りたいな」と思った瞬間からどんどんと違う風景になってしまう。
  • 自然相手の撮影は難しい。ほんと、瞬時の判断が求められる。
  • 強風で三脚もろともカメラがぶっ飛びそうだった。この点でも、自然は怖いぜよ。
  • 以上。

フォトウォーク＠お台場

お台場に行ってきたので、画像を貼っていく

  • 趣味であるカメラの写真を貼ります
  • カメラは、昨年の秋に買った Nikon D3400 です。


  • 2019年3月10日(日)に行ってきました。
  • お台場海浜公園駐車場に車を止め、アクアシティに入ろうとしたらキレイな桜が咲いていたのでパシャリ。
  • 花と葉っぱがあったので、ソメイヨシノでは無さそうです。コウヅザクラ？って言うの？


  • 買い物を済ませ、お昼を食べてからお台場海浜公園へ。
  • お昼は、デックスのカザーナというインド料理の店へ。育ち盛りの息子を持つ身としてはコスパの良いランチバイキングは助かりました。もちろん美味しかったです!!
  • モヤイ像。渋谷のとはまた違う趣のお顔。


  • カモメにパンをあげてる人がいたので、すかさずパシャリ。
  • 実際にはカモメとかなり近づけました。鳥に餌をやるときって、普通は地面や水面に餌を投げますが、カモメって投げて与えたほうがいいのか？


  • 変わった船がいました。出発前に船長さんと港の職員さん。バッグの受け渡しが、何やら怪しさを醸し出してるな、と勝手に想像。


  • こっちの船も、黒服のおっさん2人が甲板に突っ立ってる。怪しい!!とまたまた勝手に想像。


  • 水陸両用バスって流行ってるの？帰りに首都高で走ってる姿も見かけました。


フォトウォークを終えて

良かった点

  • 初めて動物の撮影に成功
    • SSを1/500にすれば、特に問題なく撮影できた
    • 運動会とかの設定と同じかな

反省点

  • 曇りの日の設定が決まらず
  • ズームレンズを車に置き忘れてしまった…
    • 広角写真はiPhoneに任せるとして、まずはズームとするか

感想

  • 年が明けてから、始めてまともにカメラを取り出して外出。
  • やっぱり曇ってる日は、思うような写真が撮れないなぁ。
  • なんか、こう、キラキラしてる感じがほしいんだよね…。
  • 以上。
